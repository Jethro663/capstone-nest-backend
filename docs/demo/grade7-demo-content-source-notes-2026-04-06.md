# Grade 7 Demo Content Source Notes (English + Science)

Date: 2026-04-06  
Scope: Nexora public `/demo` tutorial content only (non-official academic records)

## Use Policy
- Content in the demo is originalized and simplified for showcase flow.
- Sources were used for curriculum alignment and topic framing only.
- Questions were rewritten and structured specifically for the Nexora demo narrative.

## English Track Reference Themes
- Reading comprehension fundamentals: main idea vs supporting details.
- Vocabulary context clues: definition/synonym/antonym clues and word-part strategy.
- Paragraph writing structure: topic sentence, coherence/transitions, conclusion sentence.

Reference anchors:
- Philippine K-12 English curriculum competencies (Grade 7 communication and reading skills).
- Open educational reading strategy resources for middle-school learners.

## Science Track Reference Themes
- Scientific method and variable control in simple investigations.
- Ecosystem interactions: biotic/abiotic factors, food chains/webs, ecological balance.
- Cell theory and basic organelle functions; plant vs animal cell distinctions.

Reference anchors:
- Philippine K-12 Science curriculum competencies (Grade 7 life science and inquiry skills).
- Open educational middle-school science references on scientific inquiry, ecosystems, and cells.

## Demo Constraints Applied
- Exactly 2 subjects: English and Science.
- Each subject has 3 modules.
- Each module has 3 lessons and 1 assessment of 5 items.
- Quarter exam has 15 harder, non-reused questions per subject.
- Threshold logic implemented for demo branching:
  - `Excellent`: >=90
  - `Good Job`: 75-89
  - `Nice Try`: <=74 (routes to teacher-side AI plan simulation + LXP branch)

## Academic Integrity Note
- Demo outputs are illustrative and are not official class records.
- AI-plan and LXP demo progression are for guided explanation of system behavior, not formal grading.
