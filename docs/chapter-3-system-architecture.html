End-to-End Flow Summary

1. User enters through web or mobile interface
2. Request is authenticated and handled by NestJS modules
3. LMS, assessment, and intervention logic is processed
4. AI tasks are delegated to FastAPI and Ollama
5. Events and queues trigger performance and notifications
6. Results persist in PostgreSQL, Redis, and shared storage
