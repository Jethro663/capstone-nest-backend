Nexora is a capstone LMS/LXP for Gat Andres Bonifacio High School. Backend is NestJS + Drizzle + PostgreSQL + Redis/Bull/EventEmitter; frontend is Next.js 16 + React 19; there is also an Expo mobile app. AI is split into a NestJS ai-mentor proxy module and a Python FastAPI ai-service for JAKIPIR chat and PDF extraction. Existing domain modules already cover classes, lessons, assessments, uploaded files/library, class record, performance snapshots/logs, LXP intervention cases/assignments/progress, analytics, reports, notifications, and audit logs.
